# NHANES 2020 — Body Measurements Analysis

A data science capstone project performing statistical analysis on the 
National Health and Nutrition Examination Survey (NHANES) 2020 dataset 
using pure NumPy and Matplotlib (no pandas).

## 📌 Key Analyses
- Weight distribution histograms and box plots (Male vs Female)
- Descriptive statistics: mean, median, skewness, kurtosis
- BMI computation and WHO category classification
- Z-score standardisation
- Scatterplot matrix with Pearson & Spearman correlations
- Waist-to-Height and Waist-to-Hip ratio analysis
- Extreme BMI participant profiling

## 🛠️ Tech Stack
Python · NumPy · Matplotlib · SciPy · Jupyter Notebook

## 📂 Dataset
NHANES 2020 via [Marek Gagolewski's teaching data](https://github.com/gagolews/teaching-data)
